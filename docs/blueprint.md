# **App Name**: StudyPlanAI

## Core Features:

- User Profiling Questionnaire: A fixed questionnaire (approx. 10 questions) gathering info about the student's time availability, workload, university schedule, current understanding of topics, and other relevant factors to help determine the optimal study plan.
- Interactive Schedule Generation: LLM tool analyzes the user profile questionnaire and constructs an interactive and personalized study schedule or timetable, incorporating topics of study and dedicated time slots.
- Schedule Customization: User is able to adjust the suggested study times, subjects and tasks to accommodate real-world scheduling conflicts or preferences.
- Downloadable PDF: The interactive, custom schedule can be downloaded as a PDF document, providing a portable and printable format.

## Style Guidelines:

- Primary color: Deep purple (#6750A4), representing intellect, focus, and sophistication.
- Background color: Light lavender (#E9DFF7), a desaturated tint of the primary for a calm and distraction-free setting.
- Accent color: Soft blue (#A0CFD6), to provide visual contrast without being intrusive.
- Body and headline font: 'PT Sans' for a readable and friendly appearance.
- Material design principles to give a layered look
- Clean, minimal UI elements with ample whitespace to avoid feeling overwhelming.
- Subtle animations and transitions to enhance the user experience without being distracting.
